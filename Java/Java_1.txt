프로그램
high level의 언어
장점 : 빠른 개발, 쉬운 보수
단점 : 완벽한 성능을 기대하기 어려움ㅜ 개발자의 지식 수준이 상대적으로 낮음

*객체 지향: 
object= oriented
객체 지향 프로그램을 작성하는데에 자바만한 언어도 없음
클래스와 상속을 지원하면서, C++과는 다른 방식으로 진행가능

*동적인 성능 확장 제공
서버로부터 동적으로 다운로드 되어 웹 브라우저에 실행 가능

* 멀티 쓰레드 지원
자바는 쓰레드라는 기능을 언어 차원에서 제공함
쓰레드란 프로그램 안에서 독립적으로 실행되는 작은 실행 단위라고 할수 있음
메모리의 공유가 가능해지면서 프로그램보다 효율적으로 실행될수 있도록 해줌
수행시간도 빠르며, 요청에 응답하는 시간도 빨라짐

* 자동 메모리 관리
자바 가상머신
안토츠리터와 같은 자바 프로그램의 실행환경을 가상적인 운영체제라는 뜻에서 자바 가상머신
