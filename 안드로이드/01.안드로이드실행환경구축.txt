뷰: 
 - 화면에 보이는 가각의 것들
 - 컨트롤이나 위젯이라 불리는 구성요소
 - 화면을 구성하는 모든것들

뷰그룹(view Group)
 - 뷰 들이 여러개 포함되어있는 그룹
 - 화면에 보이지 않는 것 까지 포함
 - 뷰 그룹안에는 뷰들이 포함된다

위젯(widget)
 - 뷰 중에 일반적으로 컨트롤의 역할을 하고 있는 것들
 - 버튼이나 텍스트 아이콘
 - 컨트롤이라 하면 눈에 보이는것들

레이아웃(Layout)
 - 뷰 그룹중에서 내부에 뷰들을 포함하고 배치하는 역할
 - 

1. Constraint Layout
2. Relative Layout
3. Linear Layout
4. Table Layout
5. FrameLayout
6. GridLayout
