Input 테그

  • form 테그 내부에서 데이터를 담는 역할
  • name: 전송한 데이터의 key값
  • type: 어떤 방식으로 입력받을지 설정한다
  • value: 원한다면 값을 지정할수 있다(미리 입력해놓는 용도)
  • placeholder: 무엇을 입력할지 미리 알려주는 용도
  • required:
  • checked:
  • disabled:
  • readonly:

type =text

제목

일반적인 입력필드를 생성한다. type을 지정해주지 않으면 text타입으로 설정


text type: 기본타입.

type =password

비밀번호

사용자의 입력을 가려주는 비밀번호 필드 required를 입력하면 무조건 입력이 되어야 전송이 된다.


누르면 전송되는 내용

type = submit

누르면 서버로 데이터를 전송하는 버튼을 생성
현재 속한 form데이터 전부를 들고 서버 페이지로 요청을 보낸다

type = reset

현재 속한 form데이터를 초기화

버튼을 누르면 초기화 되는 내용

type = radio

사용자가 한가지만 선택할 수 있게 만듬
같은 name으로 설정하면 한가지만 선택할 수 있게 된다.




type = checkbox

사용자가 여러가지를 선택할 수 있다 name속성을 같은 값으로 설정하면 선택한 모든 값을 **배열로 보낸다 checked 속성을 설정하면 미리 체크됨.
